ABR (Accelerated Benefits Riders) são o **diferencial competitivo da NLG** — todos sem custo adicional em IULs e WL. Permitem cliente acessar parcial do death benefit ainda em vida quando critérios médicos são atingidos.

**Set completo NLG:**

1. **Terminal Illness Rider** — paga até **$1.5M lifetime cap**. Triggered se médico certifica que cliente tem expectativa de vida ≤24 meses (varia por estado). Prerequisito de acesso: diagnóstico oficial.

2. **Chronic Illness Rider** — paga quando cliente não consegue executar **2 of 6 ADLs (Activities of Daily Living) por ≥90 dias** OU sofre **cognitive impairment severo**. ADLs: bathing, continence, dressing, eating, toileting, transferring.

3. **Critical Illness Rider** — paga em diagnóstico de condições qualificadas: ALS, câncer, AVC, infarto, Major Organ Transplant, kidney failure, etc. **Cap $1M lifetime**.

4. **Critical Injury Rider** — paga em injuries graves (paralysis, severe burns, coma, etc).

5. **Alzheimer's Disease Rider** (industry-first 2023) — paga em diagnóstico de Alzheimer ou Lewy Body Dementia. Esse é diferencial NLG — pouca carrier oferece como rider explícito.

6. **Fertility Journey Rider** (industry-first 2024) — paga em despesas de fertilidade (IVF, treatments). Outro diferencial recente da NLG.

**Disponibilidade por produto:**

- **FlexLife / FlexLife NL:** ABR set completo sem custo.
- **PeakLife / PeakLife NL:** ABR set completo sem custo.
- **SummitLife / SummitLife NL:** ABR set completo + Estate Preservation Rider.
- **SurvivorLife / SurvivorLife NL:** ABR set completo (na primeira morte? — confirmar lógica de SIUL).
- **TotalSecure / TotalSecure NL:** ABR set completo (em WL!).
- **BasicSecure / BasicSecure NL:** ABR sem custo.
- **Term LSW / Term NL Life:** ABR set completo.
- **RapidProtect / RapidProtect NL:** ABR set NÃO disponível em **Express Standard classes**.

**Restrições por classe (importante):**

- Express Standard NT 2 (rating 225–300%) — **LBR e LIBR NÃO disponíveis**.
- RapidProtect Express classes — ABR NÃO disponível.
- Foreign Nationals — ABR disponíveis apenas em **A & B countries**.

**Como funciona o pagamento (importante explicar ao cliente):**

- Acceleration **reduz death benefit** quando triggered (não é benefit adicional).
- Reduction calculation: amount accelerated × discount factor (varia por idade + condition).
- Reserve continua acumulando após acceleration parcial.
- Tax treatment: terminal illness é geralmente **tax-free**; chronic illness sob certas condições (HIPAA-qualified). Sempre dizer cliente pra consultar CPA.

**Posicionamento de venda:**

- ABRs são razão #1 pra escolher NLG sobre concorrência (Allianz/Penn/Lincoln) em IUL/WL.
- Mostrar ABR set completo no pitch como "living benefits sem custo extra".
- Pra cliente jovem: enfatizar Critical Illness/Injury (mais provável que terminal nesse tier).
- Pra cliente 55+: enfatizar Chronic Illness (long-term care alternative).
- Pra cliente com family history Alzheimer: rider específico é unique selling point.

**Pitfalls:**

- **Não esqueça de offer ABR explicitamente** — pitfall #13 do plano. Diferencial NLG vs concorrência.
- Express classes (substandard) podem perder acesso → comunique ao cliente antes de cotar.
